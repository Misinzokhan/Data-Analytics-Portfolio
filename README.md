# 📊 Data Analytics Portfolio

## 🔹 Projects

### 1. Superstore Sales Analysis
**Objective:** Analyze sales performance and profitability across regions and categories.  
**Tools:** MySQL, Power BI  
**Key Insights:**
- Identified top-performing regions and products  
- Detected profit/loss patterns across categories  
- Highlighted sales trends over time  

---

### 2. Hospital Operations Analysis
**Objective:** Evaluate hospital performance using operational and patient data.  
**Tools:** Python (EDA), Power BI  
**Key Insights:**
- Patient flow and peak load periods  
- Revenue distribution by department  
- Relationship between waiting time and patient satisfaction
